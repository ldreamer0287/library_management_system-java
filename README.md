Library Management System
🔷 Description
This is a simple GUI-based Library Management System developed using Java Swing. It allows users to manage books and members, issue and return books, and search records.

The system uses file handling to store data in text files.

🚀 Features
Add Book
Add Member
Issue Book
Return Book
Search Book & Member
Show All Records
Data stored using text files
Technologies Used
Java
Java Swing (GUI)
File Handling
📂 Files
MainFrame.java → GUI design and logic
FileHandler.java → File read/write
Start.java → Main class
books.txt → Book list
members.txt → Member list
issuedBooks.txt → Issued books
▶️ How to Run
Open terminal
Compile:
javac Start.java
Run:
java Start
